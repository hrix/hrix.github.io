travis-ciやめてGithub actionsでhexoビルド

投稿日 2023-03-07 編集日 2025-09-16

Github Pages

これまで

  • hrix.github.ioリポジトリのブランチでで投稿用とサイト用（master)で分けていた
  • ビルドするのにはtravis-ciを使っていた

変わったこと

  • travis-ciが使えなくなった
    • travis-ci時代のブランチも参考のため残している
  • Github actionsが使うように変更
  • 投稿用リポジトリ（private）とサイト用リポジトリを別にした
  • jekyllの方がgithub actions と親和性あるようだが、nodejsがいいのでhexoを使う

参考

  • https://yatakemi.github.io/2021/08/13/hexo-deploy-github-actions/
  • https://docs.github.com/ja/pages/getting-started-with-github-pages/about-github-pages
